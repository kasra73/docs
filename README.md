# راهنمای کاربردی کوبرنتیس!

این کتاب آنلاین، با هدف کمک به جامعه فارسی و تسریع فرآیند یادگیری استفاده از تکنولوژی کوبرنتیس توسط علاقه‌مندان گردهم‌آوری شده‌است! پس شما نیز می‌توانید در این‌کار سهیم باشید و همچنین صد البته این پروژه، با کمک شما عزیزان پابرجا خواهد ماند!

جهت کمک کافی‌است که [مخزن کتاب](https://github.com/kubernetes-fa) را در گیت‌هاب فورک کنید و پس از ایجاد تغییرات، یک `pull request` به مخزن اصلی بفرستید تا پس از بازبینی و تایید، به مخزن اصلی اضافه گردد. برای توسعه این کتاب، از ابزار [gitbook](https://www.gitbook.com/) استفاده می‌کنیم که از فرمت مارک‌داون استفاده می‌کند و همچنین روش نصب و راه‌اندازی آن به صورت لوکال در ریپو گیت‌هاب آنها موجود می‌باشد.
